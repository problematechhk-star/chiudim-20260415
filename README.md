# 觀潮浸信會「潮食工房」會員申請表

這是一個基於 React (Vite) 和 Express 的會員申請表系統。支持本地 CSV 數據存儲、PDF 生成以及移動端 WiFi 訪問。

## 快速開始 (在新電腦上運行)

在另一台電腦上運行此項目，請確保已安裝 [Node.js](https://nodejs.org/) (建議 v18 或更高版本)。

### 1. 下載項目
從 GitHub 克隆項目到本地：
```bash
git clone https://github.com/problematechhk-star/chiudimform.git
cd chiudimform
```

### 2. 安裝依賴
在項目根目錄運行：
```bash
npm install
```

---

## 🚀 一鍵啟動 (推薦)

為了方便非開發人員使用，項目提供了「一鍵啟動」腳本，會自動安裝依賴並同時啟動前後端服務。

### macOS 用戶
1. 在 Finder 中找到 **`啟動服務.command`**。
2. **雙擊** 該文件即可啟動系統。
3. *第一次運行可能需要幾分鐘來下載必要的組件。*

### Windows 用戶
1. 在文件資源管理器中找到 **`啟動服務.bat`**。
2. **雙擊** 該文件即可啟動系統。

---

## 手動啟動 (開發人員)

如果不想使用腳本，可以手動啟動：

### 1. 啟動後端伺服器 (保存數據用)
打開一個終端機，運行：
```bash
node server.cjs
```
伺服器將在 `http://localhost:3001` 運行。數據將保存在 `database/` 文件夾中。

### 2. 啟動前端表格
打開另一個終端機，運行：
```bash
npm run dev
```
前端將在 `http://localhost:8080` 運行。

## 功能說明
- **數據庫**: 所有填寫的資料都會自動保存到 `database/database.csv`。
- **PDF 檔案**: 提交後生成的 PDF 申請表會保存在 `database/pdf/`。

### 手機訪問
只要手機和電腦連接同一個 WiFi，就可以用手機填寫表格：
1. 在電腦終端機查看電腦的 IP 地址 (例如 `192.168.68.55`)。
2. 在手機瀏覽器輸入 `http://電腦IP:8081` (例如 `http://192.168.68.55:8081`)。
3. 點擊頂部的 **「測試伺服器連接」** 確保手機可以成功連接到電腦。

### 備份與安全
由於 `.gitignore` 的設置，`database/` 文件夾內的私隱數據 **不會** 被上傳到 GitHub。請記得定期手動備份電腦上的 `database/` 文件夾。

---

## ⚠️ 重要注意事項 (必讀)

為了確保數據能正確儲存，請在使用系統時遵守以下規則：

1. **關閉 Excel 檔案**：
   - 在使用本系統進行「舊會員查詢」或「提交表單」時，請**務必關閉**電腦上已打開的 `會員資料template.xlsx` 或 `database/database.csv` 檔案。
   - **原因**：如果 Excel 正在開啟這些檔案，系統將無法寫入新數據，這會導致 **PDF 產生空白** 或 **數據儲存失敗**。

2. **確保伺服器正在運行**：
   - 提交表單前，請確認「啟動服務」的終端機視窗沒有被關閉。
